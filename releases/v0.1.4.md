v0.1.4 — Docker Desktop OTA mount fix · Исправление mount при OTA на Docker Desktop

---

## English

### Fixed

- OTA deploy on Docker Desktop (Mac/Windows): `mounts denied: /deploy/infrastructure/...` when running `docker compose` from the update container
- All bind mounts and build contexts now use `PYORCH_HOST_PROJECT_ROOT` for host-resolved paths

**Full Changelog**: https://github.com/PyOrchestrator/PyOrchestrator/compare/v0.1.3...v0.1.4
