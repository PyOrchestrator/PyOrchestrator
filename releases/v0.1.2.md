v0.1.2 — Settings layout and OTA executor fixes · Вёрстка Settings и исправления OTA executor

## English

### Added
- Equal-height responsive layout for Profile, Localization, and Appearance cards on Settings
- Recent failed update attempts shown in the Updates panel

### Fixed
- OTA update executor: Docker CLI in backend image, correct Compose volume/network names, host project path (`PYORCH_HOST_PROJECT_ROOT`)
- Update scheduler: Postgres advisory lock prevents duplicate auto-updates on startup

**Full Changelog**: https://github.com/PyOrchestrator/PyOrchestrator/compare/v0.1.1...v0.1.2
