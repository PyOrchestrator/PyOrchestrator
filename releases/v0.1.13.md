## English

### Changed

#### Backend dependencies
- uvicorn 0.49.0
- alembic 1.18.5
- python-jose 3.5.0
- psutil 7.2.2 (backend + runtime image)

#### Documentation
- CHANGELOG, release notes, README, wiki, and GitHub Pages docs synced with current stack
- Stack tables updated (Uvicorn, Alembic, react-router-dom 7, psutil 7)

### Upgrade

Use **Settings → Software Updates** in the admin panel, or:

```bash
git fetch --tags
git checkout v0.1.13
docker compose -f docker-compose.yml -f docker-compose.prod.yml up -d --build
```

See [v0.1.12](https://github.com/PyOrchestrator/PyOrchestrator/releases/tag/v0.1.12) for script API fixes and runtime Redis resilience.
