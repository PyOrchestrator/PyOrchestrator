## English

First stable release of **PyOrchestrator** — SCADA/CMS for managing isolated Python scripts and bots in Docker Compose.

### Highlights

- **Control plane:** FastAPI + React UI, JWT/RBAC (4 roles), Monaco editor, dashboard
- **Runtime:** subprocess sandbox + venv + rlimits, Redis queue, live logs
- **Scheduler:** cron, intervals, webhooks (APScheduler)
- **Data:** PostgreSQL, MinIO, encrypted secrets vault, backups
- **Observability:** Prometheus, Grafana, Loki
- **AI agents:** MCP server (24+ tools, HTTP + stdio)
- **Docs:** https://pyorchestrator.github.io/PyOrchestrator/en/

### Quick start

```bash
git clone https://github.com/PyOrchestrator/PyOrchestrator.git
cd PyOrchestrator
git checkout v0.1.0
cp .env.example .env
docker compose up --build
```

**Default login:** `admin@pyorchestrator.local` / `admin`
