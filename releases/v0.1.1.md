## English

### Highlights

- **OTA updates** from GitHub Releases — check, apply, rollback, auto-update
- **UpdateBanner** + redesigned Settings panel for administrators
- **Notifications** when updates are available or in progress
- Detached Docker updater with health check and automatic rollback

### Test the update flow

1. Run **v0.1.0** (or set `APP_VERSION=0.1.0` in `.env`)
2. Open **Settings → Software updates** → **Check now**
3. You should see **v0.1.1** available → **Update now**

## Русский

### Основные изменения

- **OTA-обновления** из GitHub Releases — проверка, установка, откат, автообновление
- **UpdateBanner** и обновлённая панель Settings для администраторов
- **Уведомления** о доступных и текущих обновлениях
- Отдельный Docker updater с health-check и автоматическим откатом

### Проверка обновления

1. Запустите **v0.1.0** (или `APP_VERSION=0.1.0` в `.env`)
2. **Настройки → Обновления ПО** → **Проверить сейчас**
3. Должен появиться **v0.1.1** → **Обновить**
