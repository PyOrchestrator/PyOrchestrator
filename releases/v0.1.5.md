v0.1.5 — Auto-detect OTA host project path · Автоопределение пути проекта для OTA

---

## English

### Changed

- OTA host project path is auto-detected from the `/deploy` volume — works on any PC without manual path configuration
- `PYORCH_HOST_PROJECT_ROOT` is now an optional override only (documented in `.env.example`)

**Full Changelog**: https://github.com/PyOrchestrator/PyOrchestrator/compare/v0.1.4...v0.1.5
