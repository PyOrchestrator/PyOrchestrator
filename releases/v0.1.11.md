## English

### Added

#### Optional Grafana observability
- Dashboard **Observability** block appears only when Grafana responds to a health check
- Configure via `GRAFANA_PUBLIC_URL` and `GRAFANA_INTERNAL_URL`

#### Optional MinIO web console
- `MINIO_CONSOLE_ENABLED=false` by default — S3 API on port 9000 works without the console
- Set `MINIO_CONSOLE_ENABLED=true` to expose the web console on port 9001

### Changed

- System page: storage bucket and MinIO status cards span full width when console is disabled
- Documentation updated for OTA updates and optional observability/storage UI

### Upgrade

```bash
git fetch --tags
git checkout v0.1.11
docker compose -f docker-compose.yml -f docker-compose.prod.yml up -d --build
```

## Русский

### Добавлено

#### Опциональная Grafana
- Блок **Наблюдаемость** на дашборде — только если Grafana отвечает на health-check
- Настройка через `GRAFANA_PUBLIC_URL` и `GRAFANA_INTERNAL_URL`

#### Опциональная MinIO Console
- `MINIO_CONSOLE_ENABLED=false` по умолчанию — S3 API на порту 9000 работает без консоли
- `MINIO_CONSOLE_ENABLED=true` открывает веб-консоль на порту 9001

### Изменено

- Страница Система: карточки bucket и статуса MinIO на всю ширину, если консоль отключена
- Документация обновлена под OTA и опциональную наблюдаемость/хранилище

```bash
git fetch --tags

docker compose -f docker-compose.yml -f docker-compose.prod.yml up -d --build
```
